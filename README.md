#*SalesIQ – Sales Analytics Dashboard*

SalesIQ is a full-stack web application that analyzes sales data from uploaded files and generates insights, charts, and forecasts.

The application allows users to upload CSV, Excel, or PDF sales data and view interactive analytics dashboards built with modern web technologies.

Features

Upload sales datasets (CSV / Excel / PDF)

Automatic data processing using Pandas

Interactive charts and visualizations

Sales forecasting using scikit-learn

Revenue anomaly detection

Dashboard UI with multiple analytics tabs
